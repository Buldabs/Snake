# 🐍 Змейка

Классическая игра «Змейка» с системой уровней, сохранением рекордов и адаптивным дизайном.  
Реализована на чистом JavaScript с использованием Canvas API.

---

## 🧰 Стек технологий

- **HTML5** — структура страницы
- **CSS3** — стилизация и адаптивность
- **JavaScript (ES6)** — игровая логика
- **Canvas API** — рендеринг графики
- **localStorage** — сохранение рекордов

## 🎮 Особенности

- Управление стрелками `↑` `↓` `←` `→`
- Рестарт по клавише `R` или кнопке
- 4 уровня сложности (скорость растёт)
- Сохранение рекорда в localStorage
- Адаптивный дизайн под мобильные устройства
- Визуальные эффекты (свечение, градиенты, глаза у змейки)

## 📂 Как запустить локально

1. Склонируйте репозиторий:
   ```bash
   git clone https://github.com/Buldabs/Snake.git

2. Установите зависимости для бэкенда
   ```bash
   cd backend
   pip install -r requirements.txt
   ```
   # или если у вас Ubuntu/WSL:
   ```bash 
   sudo apt install python3-flask python3-flask-cors -y1 
   ```


3. Запустите сервер
   ```bash
   python3 app.py
   ```

4. Откройте игру в браузере
Перейдите по адресу: http://localhost:5000
