# 🐍 Змейка

Классическая игра «Змейка» на чистом JavaScript.

## Стек
- HTML5 + CSS3
- JavaScript (ES6)
- Canvas API

## Как запустить локально
1. Скачайте репозиторий
2. Откройте `index.html` в браузере
